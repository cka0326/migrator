For Each table Metadata needed:
- table_name: name of the table; (User given text)
- name_space: name space in which this table is stored; (User given text)
- description: description of the table; (User given text)
- Environment: DEV, TEST, UAT, PROD; (dropdown of these values)
- business_domain: Example: Claims, Policy, Billing, Finance, etc; (User given text)
- row_count: count of rows in the table; (User given number)
- column_count: count of columns in the table; (User given number)
- has_primary_key: Dropdown of boolean values;
- unique_key_columns: comma separated list of column names; (User given text)
- grain_description: description of grain of the table; (User given text)
- refresh_frequency: Daily, Weekly, Monthly, Ad-hoc; (dropdown of these values)

For Each column Metadata needed:
- column_name: name of the column; (User given text)
- table_name: name of the table in which column exist; (User given text)
- name_space: name space in which table exist; (User given text)
- data_type: Data Type of the column; (User given text)
- null_count: count of null values in the column; (User given number)
- max_length: maximum length of the column; (User given number)
- max_value: maximum value in the column; (User given text)
- min_value: minimum value in the column; (User given text)
- precision: precision of the column; (User given number)
- nullable: boolean value; (Dropdown of boolean value)
- default_value: Default value of the column; (User given text)
- column_definition: definition of the column; (User given text)
- column_computation_formula: formula for computing the column; (User given text)
- unique_count: count of unique values in the column; (User given number)
- uniques: List of unique values in the column; (User given text comma separated)
- mean_value: mean value of the column; (User given number)
- stddev_value: standard deviation of the column; (User given number)
- sum_value: sum of values in the column; (User given number)
