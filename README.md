# Migrator

This repository contains **Lineage Canvas**, a local-first web app for mapping and analyzing data lineage through system migrations (e.g. SAS → Snowflake).

The application lives in the [`lineage-canvas/`](./lineage-canvas) directory.

## Quick start

```bash
cd lineage-canvas
npm install
npm run dev
```

Then open the URL Vite prints (typically http://localhost:5173).

📖 **Full documentation** — installation, usage, import formats, comparison/analysis, and troubleshooting — is in [`lineage-canvas/README.md`](./lineage-canvas/README.md).
